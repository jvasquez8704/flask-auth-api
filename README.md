# katch-auth-api

# Enviroment
    * Python: 3.8.4 || 2.7.16

# Dependencies
    * run `$ pip3 install Authlib`
    * run `$ pip3 install Authlib requests` 
    * run `$ pip3 install Authlib Flask`
    * run `$ pip3 install Flask` 
    * run `$ pip3 install request` 
    * run `$ pip3 install firebase_admin`
